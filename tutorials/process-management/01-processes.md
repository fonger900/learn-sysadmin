# Viewing Processes

Every program running on your system is a process.

## The `ps` Command
Shows a snapshot of running processes.
- `ps`: Processes running in current shell
- `ps aux`: Detailed list of ALL processes on system
  - `a`: All users
  - `u`: User-oriented format
  - `x`: Processes without terminal

## The `top` Command
Real-time view of system processes and resource usage.
- Shows CPU, Memory usage, Load Average.
- Press `q` to quit.

## The `htop` Command
A more user-friendly, colorful alternative to `top` (needs to be installed).
