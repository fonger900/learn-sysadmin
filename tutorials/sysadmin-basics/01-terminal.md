# Terminal Basics

## The Command Line Interface (CLI)
The CLI is your primary tool as a sysadmin. Unlike a GUI (Graphical User Interface), it allows you to interact with the system using text commands.

## Opening the Terminal
- **GNOME**: press `Ctrl` + `Alt` + `T`
- **Mac**: `Cmd` + `Space`, type "Terminal"

## The Prompt
You will see something like `user@hostname:~$`.
- `user`: Your current username
- `hostname`: The machine's name
- `~`: Current directory (home)
- `$`: Standard user prompt (`#` indicates root)

## Basic Navigation
- `pwd`: Print Working Directory
- `ls`: List files
- `cd`: Change Directory

```bash
$ pwd
/home/user
$ ls
Desktop Documents Downloads
$ cd Documents
```
